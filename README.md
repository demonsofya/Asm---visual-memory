# Asm---visual-memory
Визуальная память асм - рамка для заданного текста

Программа реализует рамку для строки, которая делится по 10 символов на одну строку

До конца строки все автоматически дополняется сердечками


Программа принимает на вход строку, содержащую несколько двух-шестисимвольных чисел 

Первое число - Тип рамки

## Первый тип:
состоит из кольцевой рамки, принимающей на вход следующие аргументы - 

    цвет символа и фона рамки / номер строки для вывода / "ширина" рамки - количество колец рамки / цвет символа и фона пользовательской строки / стиль рамки - число от 01 до 04
<img width="1046" height="682" alt="image" src="https://github.com/user-attachments/assets/2f9483a4-44ec-4429-b5bc-af11c2900c6b" />
    стиль рамки состоит из следующих вариантов:
    1. стрелочки
    2. масти
    3. смайлики и ноты
    4. цифры от 1 до 4

Также печатает строку с помощью для пользователя (повторение принимаемых аргументов)
    
## Второй тип:
состоит из прямоугольной рамки с окантовкой (вдохновленно рамкой для shift + f4)

принимает на вход следующие аргументы:

      цвет символа и фона рамки / высота рамки / ширина рамки / цвет символа и фона пользовательской строки / номер строки

## Третий тип:
состоит из цифр от 1 до 9

      цвет символа и фона рамки / высота рамки / ширина рамки / цвет символа и фона пользовательской строки / номер строки
